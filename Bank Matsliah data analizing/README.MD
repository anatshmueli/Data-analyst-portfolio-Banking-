# 🏦 Bank Matsliakh – End-to-End Banking Data Analytics Project

## 📖 Project Overview

**Bank Matsliakh** is an end-to-end data analytics project that simulates a real banking environment using **Microsoft SQL Server** and **Python**.

The project demonstrates the complete data analysis lifecycle, including relational database design, SQL querying, data processing, statistical analysis, and business visualization.

A relational banking database was designed and populated with transactional and customer data to analyze banking operations between **2018 and 2023**. The project focuses on identifying customer behavior patterns, branch performance, deposit activity, customer complaints, and account closures.

The objective is to transform raw banking data into meaningful business insights that support informed decision-making.

# 🏦 Business Scenario

Banks collect large volumes of customer and financial data every day. Transforming this data into actionable insights is essential for improving operational efficiency, customer satisfaction, and business performance.

This project simulates a real banking environment and analyzes customer activity over a six-year period to answer important business questions regarding account activity, branch performance, customer retention, and financial behavior.

# 🗄 Database Design

The project is based on a relational SQL Server database containing six connected tables.

| Table | Description |
|--------|-------------|
| 👨‍💼 **Bankers** | Stores bank employee information |
| 👥 **Clients** | Stores customer information |
| 💳 **Accounts** | Stores customer Current and Savings accounts |
| 💰 **Deposits** | Stores customer deposits |
| 🏦 **Loans** | Stores loans and mortgages |
| 💳 **Cards** | Stores issued bank cards |

The database was designed using relational database principles, including:

- Primary Keys
- Foreign Keys
- Referential Integrity
- Constraints
- One-to-Many Relationships
- Database Normalization
- Entity Relationship Diagram (ERD)

# 📈 Key Findings

The analysis identified several important business insights:

- **2019** recorded the highest number of newly opened bank accounts.
- Branches **12** and **92** achieved the strongest performance.
- Deposit activity reached its highest level during **Q4 2022**.
- Account closures increased significantly during the last quarter of **2023**.
- A **moderate positive correlation (0.435)** was identified between customer complaints and account closures, suggesting that customer satisfaction may influence customer retention, while additional external factors may also contribute to customer churn.
